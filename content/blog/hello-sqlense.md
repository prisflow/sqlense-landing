## 为什么需要 SQLense？

数据库课程是计算机专业的核心基础课，但实验教学长期面临几个痛点：

- **环境配置繁琐**：每个学生都需要安装数据库客户端、IDE，不同操作系统兼容性问题层出不穷
- **教师监控困难**：实验课上教师难以实时掌握每位学生的进度，课后批改 SQL 作业耗时巨大
- **反馈不及时**：学生写错了 SQL 要等到下次课才能得到反馈，学习效率低下

SQLense 正是为了解决这些问题而生。

## 我们的解决方案

### 开箱即用的云端 IDE

基于 code-server 构建，学生打开浏览器即可获得完整的 VS Code 开发体验。预装 SQLTools 扩展，支持 PostgreSQL 连接、SQL 智能补全、查询结果可视化。

### 实时教学监控

教师仪表盘展示全班学生的在线状态、SQL 执行记录、操作时间线。支持远程画面查看（Screen Takeover），随时了解每位学生的实际操作情况。

### AI 辅助分析

利用大语言模型（DeepSeek / 混元）自动分析学生的 SQL 语句：

- 检测语法错误与逻辑缺陷
- 分析查询效率，给出索引优化建议
- 识别常见教学误区，生成个性化反馈
